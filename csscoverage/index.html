CSS Coverage Demo

Chrome DevTools:

  1. Open Chrome DevTools
  2. Click on the Audits tab within the Developer Tools window
  3. Ensure the Web Page Performance option is checked.
  4. Click Run
  5. Load pages following the links of the list above (points 3)
  6. Stop the audit

Others:

  1. Open the developer toolbar (Shift+F2)
  2. Type "csscoverage start"
  3. Click through the pages in turn.
    Next: Page 1
  4. When you're done collecting data, close the notification bar or type "csscoverage stop"
  5. In the style editor check for selectors with a strikeout, or type "csscoverage report"
